Battle of the Ages

Battle of the Ages is an Android battle card game that uses a NIOSII microprocessor to link multiplayers into one gameplay session. Together with three other students, we built this game under the guidance of Professor Thor during our ECE381 course.

To build Battle of the Ages, we had to planned and designed different aspect of the game such as multiplayer connectivity, audio playability through the NIOSII processor, event synchronization and android application programming. I contributed in managing audio playability and the front-end interface design which includes both programming and designing images.

Child First Reporting System

Child First is a website that serves to improve the accessibility of reporting child abuse cases to the appropriate child protection services. Together with Kota Takaoka Reserach Lab, the website was built to give further support to professional social workers at Japan and Vancouver to simplify all child abuse reporting processes.

To build Battle of the Ages, we had to planned and designed different aspect of the game such as multiplayer The website includes a login system to separate both administrators and professional social workers accounts, a dual language support (English and Japanese), and a risk assessment sheet form that is to be submitted for any child abuse reports.

During this project, I was responsible for designing and implementing the front-end interface and also linking the website to an external server for database storage. Child First is built using AngularJS (Javascript, HTML/CSS) and uses PHP as a communication language between the front-end and back-end.

Olympia

Olympia is a prototype mobile application that targets audiences who are keen on keeping themselves updated with the latest news on all ongoing Olympic events. Olympia is designed as an application with a simple and intuitive interface so that users can surf around the application without hindering their busy and active lifestyle. The application focuses more on user customization to deliver Olympic news that the user is interested in to better serve the user's purpose on using the application.

Together with four other students, Olympia was designed under guidance of Professor Sid Fels for ECE 418 (Human Computer Interfaces). Olympia was designed using Ionic framework (Javascript and HTML/CSS).
